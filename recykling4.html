      <p>W sprawie recyklingu w grę wchodzi obliczanie jego efektywności. W tym celu Oblicza się procentową ilość materiałów, która jest faktycznie przetwarzana i wykorzystywana ponownie z odpadów. Wyższy odsetek recyklingu oznacza mniejszą ilość odpadów, która trafia na składowisko.

Aby obliczyć efektywność recyklingu, wykonuje się następujące kroki:

Określenie ilości materiałów poddanych recyklingowi: Pierwszym krokiem jest określenie ilości zużytych materiałów, które są gromadzone i poddawane recyklingowi. Może to obejmować zużyty papier, plastik, szkło, metal i inne materiały.

Określenie ilości przetworzonych materiałów: Następnie należy określić ilość materiałów, która została faktycznie przetworzona i wykorzystana ponownie w procesie recyklingu. To może obejmować ilość przetworzonego papieru, plastiku, szkła, metalu i innych surowców.

</p>

    <div class="card">

      <h1>Oblicz wskaźnik recyklingu!</h1>

      <p>Wskaźnik recyklingu można obliczyć, stosując następującą formułę:

Wskaźnik recyklingu = (Ilość przetworzonych materiałów / Ilość materiałów poddanych recyklingowi) x 100

Wynik jest wyrażony w procentach i reprezentuje odsetek materiałów, który został przetworzony i wykorzystany ponownie w procesie recyklingu.

</p>

<h2>Wskaźnik recyklingu</h2>

 

  <label for="total-waste">Całkowita ilość odpadów:</label>

  <input type="number" id="total-waste" min="0">

  <br>

  <label for="recycled-waste">Ilość zrecyklowanych odpadów:</label>

  <input type="number" id="recycled-waste" min="0">

  <br>

  <button onclick="calculateRecyclingRate()">Oblicz</button>

  <br>

  <div id="result"></div>

  <script>

    function calculateRecyclingRate() {

      const totalWaste = parseFloat(document.getElementById('total-waste').value);

      const recycledWaste = parseFloat(document.getElementById('recycled-waste').value);

      if (isNaN(totalWaste) || isNaN(recycledWaste)) {

        document.getElementById('result').innerHTML = "Podaj poprawne wartości.";

        return;

      }

      if (totalWaste < 0 || recycledWaste < 0 || recycledWaste > totalWaste) {

        document.getElementById('result').innerHTML = "Podaj poprawne wartości.";

        return;

      }

      const recyclingRate = (recycledWaste / totalWaste) * 100;

      const formattedRate = recyclingRate.toFixed(2);

      document.getElementById('result').innerHTML = "Wskaźnik recyklingu: " + formattedRate + "%";

    }

  </script>

      <p>Analiza i ocena efektywności: Następnie należy przeanalizować wyniki i ocenić efektywność recyklingu. Wyższy wskaźnik recyklingu oznacza większą efektywność, ponieważ większa ilość materiałów jest przetwarzana i wykorzystywana ponownie, a mniej trafia na składowiska.

</p>
